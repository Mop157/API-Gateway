запрос на инструмент:

{
    target: "название айпи или домена", // строка
    number: количество пингов (макс 10) // число
}

пример:

{
    target: "google.com",
    number: 4
}

описание:
обычная команда пинг айди или домена

авторизация: обязательно
выбор языка: не обязательно ( по стандарту EN )
